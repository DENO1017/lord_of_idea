# QA 人工测试结果日志（Test Result Log）

每次按 [APP 人工测试条目](app_manual_test_checklist.md) 执行一轮人工测试后，在此追加一条**测试结果摘要**。便于追溯版本质量与遗留问题。

---

## 记录格式说明

| 字段 | 说明 |
|------|------|
| 日期 | 测试执行/汇总日期（YYYY-MM-DD） |
| 版本/构建 | 被测包标识（如 git 短 hash、构建号，可选） |
| 测试范围 | 依据的测试条目文档与模块（如 P0+P1 全量、或 P1 工具回归） |
| 结果统计 | 通过 / 未通过或待优化 / 跳过 数量及总计 |
| 结论 | 是否通过发布/转测、阻塞项说明 |
| 遗留问题 | 未通过项编号与简要描述，或链接到问题记录表/缺陷单 |
| 待办 | 建议的复测、修复或文档更新 |

---
