# 《灵感之主》项目概览

## 项目名称

**灵感之主**（Lord of Idea）

## 项目简介

《灵感之主》是一款集成了多种占卜、骰子、诗签等创意辅助工具的移动应用。用户可以在应用中学习塔罗牌等占卜方法，并将这些工具用于**跑团**（桌面角色扮演游戏）与**故事创作**的辅助。应用核心围绕「电子手帐」展开：用户可将占卜或骰子结果插入手帐、书写解读，并用手帐常见的贴纸、胶带等装饰进行自定义，打造个性化的灵感记录本。同时支持**共享手帐**与跑团模板，方便线上跑团时在共享手帐中记录角色行动与骰子结果，DM/KP 作为手帐持有者可设置部分内容的阅读权限，用于暗骰与背景设定记录。

---

## 主壳与底部导航

应用主页面最下方为**五个页签**：**首页**、**工具**、**手账**、**市集**、**我的**。各页签的职责与详细规格见 [主壳与底部导航规格](../design/main_shell_spec.md)、[路由规格](../technical/routes_spec.md)。

- **首页**：社区功能，浏览他人分享的手账与动态，支持互动与收藏，卡片网格、流式下拉刷新与算法推荐（见 [首页规格](../design/home_screen_spec.md)）。
- **工具**：骰子、诗签、占卜等工具入口，按钮形式排布。
- **手账**：手账列表（本地手账与共享手账均在此），按时间顺序、卡片网格排布。
- **市集**：市集功能（具体能力待产品后续定义）。
- **我的**：主要为各类设置与个人相关入口。

## 核心功能模块

### 1. 占卜与随机工具

- **占卜工具**：塔罗牌等多种占卜方式
- **学习与用法**：内置占卜法教学（如塔罗牌阵、解读方法等）
- **骰子**：通用骰子工具，支持多种面数与用法
- **诗签**：抽签、诗签类随机灵感工具

**使用场景**：为跑团剧情、角色命运、故事走向提供随机灵感与参考。

### 2. 电子手帐

- **编辑与记录**：富文本/手帐式编辑，支持插入占卜结果、骰子结果
- **解读书写**：在结果旁书写个人解读、笔记与联想
- **装饰系统**：贴纸、胶带等手帐常见装饰元素，可自定义添加与排版
- **视觉呈现**：以「手帐」形式呈现，兼顾美观与实用

**使用场景**：个人灵感记录、占卜日记、创作素材整理、跑团个人笔记等。

### 3. 共享手帐与跑团支持

- **共享手帐**：多人可共同查看与编辑的电子手帐
- **跑团模板**：提供适合线上跑团的页面/结构模板
- **协作记录**：在共享手帐中记录角色行动、骰子结果、剧情节点
- **权限与隐私**：
  - 手帐持有者：DM 或 KP
  - 可对**部分内容**设置阅读权限（如仅持有者可见）
  - 用于**暗骰**（仅主持人可见的骰子结果）
  - 用于记录**背景设定**、秘密信息等

**使用场景**：线上 TRPG 跑团记录、团内信息分层、主持人专用笔记与暗骰。
